# Wine Quality Prediction

## Intern ID: CITS4322


## Project Overview

This machine learning project predicts the quality of wine based on its chemical properties such as acidity, sugar content, pH level, alcohol content, and sulphates.

The model is trained using the Wine Quality Dataset and uses the Random Forest Classifier algorithm to make predictions.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn

---

## Dataset

The dataset contains various physicochemical features of wine, including:

- Fixed Acidity
- Volatile Acidity
- Citric Acid
- Residual Sugar
- Chlorides
- Free Sulfur Dioxide
- Total Sulfur Dioxide
- Density
- pH
- Sulphates
- Alcohol

Target Variable:

- Quality

---

## Project Workflow

1. Import required libraries
2. Load dataset
3. Separate features and target variable
4. Split data into training and testing sets
5. Train Random Forest Classifier model
6. Predict wine quality
7. Evaluate model accuracy
